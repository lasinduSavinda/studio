# **App Name**: LunaCycle

## Core Features:

- Period Calendar: Mark start and end dates of period.
- Color-coded days: Color-coded days (menstruation, fertile, ovulation).
- Cycle history: Save cycle history locally.
- Cycle Prediction: Predict next period based on the rolling average of past cycles.
- Symptoms Tracker: Track mood, cramps, headaches, bloating, acne using a tool powered by generative AI to help to suggest additional potential causes for your specific ailment.
- Daily Notes: Enable users to write personal notes for each day, saved locally.
- Reminders: Enable local notifications for period start, ovulation.
- Export: Export period history and symptoms as PDF

## Style Guidelines:

- Primary color: Soft lavender (#D0B8FF) to evoke a sense of calmness and femininity.
- Background color: Light gray (#F0F0F5) to create a clean, non-distracting backdrop.
- Accent color: Pale pink (#FFD1DC) to highlight important dates and CTAs.
- Body and headline font: 'PT Sans', a humanist sans-serif for readability and modern feel.
- Simple, intuitive icons for symptoms and reminders.
- Clean, single-page layout with clear sections for each feature.
- Subtle animations for transitions and feedback to add a gentle, intuitive feel.
